# Contributing

Thanks for your interest in midnight-wallet-cli!

## About This Repo

This repository is the **documentation and issue tracker** for the [midnight-wallet-cli](https://www.npmjs.com/package/midnight-wallet-cli) npm package. Source code is not hosted here.

## How to Help

### Report a Bug

[Open a bug report](https://github.com/nel349/midnight-wallet-cli-hub/issues/new?template=bug_report.md) and include:

- **CLI version**: run `midnight --version`
- **OS and Node.js version**: e.g., macOS 15, Node 22.x
- **Command you ran**: the exact command and flags
- **Expected behavior**: what you expected to happen
- **Actual behavior**: what actually happened (include full output)

### Request a Feature

[Open a feature request](https://github.com/nel349/midnight-wallet-cli-hub/issues/new?template=feature_request.md) describing:

- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

### Improve Documentation

If you find errors or gaps in the docs, open an issue describing what's wrong or unclear. Pull requests to improve documentation in the `docs/` folder are welcome.

## Code of Conduct

Be respectful and constructive. We're all here to build on Midnight.
